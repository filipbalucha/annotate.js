annotate.js
Annotations, made effortless.

Annotate.js allows users to highlight and annotate text in the DOM. The annotations persist across sessions and can be easily navigated using the annotate.js navigator component. It's lightweight and has no dependencies â€“ not even jQuery!

See for yourself:

  1. Select some text
  2. Pick a highlight color
  3. Repeat!
  4. Click on a highlight to modify it
Current features:
  • Annotations persist across sessions (note: persisting color changes coming soon)
There's a lot more in store, so stay tuned. Upcoming features:
  • Adding comments to annotations
  • Deleting annotations
  • Annotation overview for easier navigation
